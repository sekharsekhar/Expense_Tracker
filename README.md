# 💰 Expense Tracker

A simple and user-friendly Expense Tracker web application that helps users manage their daily income and expenses efficiently.

---

## 🚀 Features

* ➕ Add income and expense transactions
* ❌ Delete transactions
* 📊 View balance, income, and expenses summary
* 💾 Data stored locally in browser (LocalStorage)
* 📱 Responsive UI design

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Storage:** Browser LocalStorage

---

## 📂 Project Structure

```
expense-tracker/
│── index.html
│── style.css
│── script.js
│── README.md
```

---

## ⚙️ How It Works

1. User enters a transaction name and amount
2. Positive amount → Income
3. Negative amount → Expense
4. Transactions are stored in LocalStorage
5. UI updates automatically with balance, income, and expense
